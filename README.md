# МДК.01.03 Разработка мобильных приложений
### Реши пример

> Создать Android-приложение которое выполняет следующие действия. Выполненное задание представить в виде ссылки на GitHub

------------

1. По нажатию на кнопку СТАРТ случайным образом генерируются два операнда и один оператор (создается числовой пример). Числа находятся в диапазоне от 10 до 99 включительно и должны быть целыми. Операции выбираются из: * (умножение), / (деление), - (вычитание), + (сумма). Фон примера - белый
2. Если осуществляется операция деление, то делимое и делитель должны быть такими, чтобы частное было целым числом
3. После генерации примера кнопка СТАРТ становится неактивной
4. Пользователь вводит в соответствующем поле ответ и нажимает кнопку ПРОВЕРКА. Кнопка должна быть активна только если неактивна кнопка СТАРТ
5. После нажатия кнопки ПРОВЕРКА поле для ввода ответа становится неактивным. Также неактивной становится кнопка ПРОВЕРКА
6. Приложение проверяет правильность ответа и сообщает результат. Если пример решен правильно, то фон примера становится зеленым, если пример решен неверно - фон примера становится красным.
7. Приложение ведет статистику правильных и неправильных ответов, которая отображается на экране, где расположен пример и кнопки
8. Также отображается процент правильных ответов. При выводе процентов значение округляется до двух знаков после запятой. Сразу после значения ставится символ "%"
